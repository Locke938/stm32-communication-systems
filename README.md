# STM32 Communication Systems

## Overview

This repository contains a collection of communication projects developed using STM32 microcontrollers.

The objective is to implement, test, and evaluate UART and I2C communication protocols in embedded systems.

---

## Technologies

- STM32
- STM32CubeIDE
- UART
- HAL Drivers
- C Programming

---

## Projects

### UART LED Control

Control STM32 LEDs from a PC through UART commands.

### UART Latency Test

Measure communication delay between STM32 and PC.

### UART Throughput Test

Evaluate UART transmission performance under different conditions.
